kali中一些工具的用法

Created2022-03-27|Updated2022-03-31|学习笔记
|Word count:1.8k|Reading time:6min|Post View:

关于kali

kali=有攻击工具的linux系统

parrot也挺好

2019之后用户名密码都是kali，之前是root和toor

默认不给root账号，sudo passwd root 创建root账号密码 live模式下重启需要重新创建

关闭自动补全功能

chsh 更改shell设定

输入密码后输入**/bin/bash**

shutdown -r now 重启

使用root权限

sudo +命令

切换root ： sudu su

passwd 给root账户设置密码

exit退回原来权限

uname -a 发现kali是基于debian，所以使用apt-get update/upgreat 更新软件

clear 清空当前终端屏幕

kali自带内部web服务器 /var/www/html/

service apache2 start

rm -rf ./* remove -recursive（循环递归） -force ./当前目录（/是根目录，危险命令） *所有

域名信息搜集

dnsenum –enum [baidu.com] 扫描子域，网段等

或者

fierce –domain [baidu.com]

不同的子域上部署不同的服务，开放不同的端口，据此来寻找漏洞

snmpcheck -t [目标] 允许用户枚举snmp设备，简单网络管理协议专门设计用于在 IP 网络管理网络节点（服务器、工作站、路由器、交换机及HUBS等）的一种标准协议

内网信息搜集

网络测试工具

dmitry -help

dmitry -wnpb [baidu.com] 参数可一起写，查询目标网站的信息

nmap -sP [内网主机] 不深入学nmap没用，有时候不靠谱，防火墙拦截

nping(nmap套件中的) -tcp -p 445(给445端口发数据包) -data AF56A43D [目标ip] 这里构造特殊data，可以刺探防火墙的反应，甚至跳过

arp-scan -l 探测内网中有哪些主机存在，并且输出mac地址，并且估计是什么设备

netdiscover -i eth0 [192.168.0.0/16 16相当于网络码占多少位，对多个网段扫描]也是使用当前网卡（ifconfig）对内网存活主机探测

masscan -p 1-65535 192.168.1.167 –rate=100 100是觉得对方可能有防火墙，跳过防火墙的效果

netbios信息搜集

通过ping计算机名可以获得东西，查到目标ip地址

文件资源管理器中，映射网络驱动器时也可以直接写计算机名字跟共享用户名即可连接到对方硬盘，而且不需要知道对方电脑密码

netbios协议可以实现计算机名变为ip地址之间的转换

nbtscan [目标ip]

nbtscan -hv [目标ip]

操作系统信息搜集

nmap [ip]

nmap -p 1-1000 [ip]

nmap 192.168.1.*

nmap -O [ip]

p0f -f /xxx/xxx/x.pcap(先抓包获得的文件) -o haha.log（对抓包进行分析输出）

msf框架

armitage图形化界面

使用木马过程见笔记

字典生成

wp站（wordpress全球用的最多的CMS）密码生成器使用

wpscan –url 网站 直接扫描wp站的用户名

cewl 网站 -w mima.txt 根据网站生成密码（这里是根据网站猜测的）

wpscan –url 网站 -U 用户名txt -P mima.txt 将上面的用户名和密码组合起来爆破，爆破出来后在wp-login.php中试一下登录

懂原理的话可以自制别的网站的工具

crunch 2 4(生成密码的位数2到4位) 0123456789 -o /root/mima.txt 生成密码

kali自带字典路径：usr share wordlists下

彩虹表生成

rtgen md5 loweralpha 1 5 0 1000 1000 0

根据提示生成彩虹表在usr/share/rainbowcrack 可以修改其中的charset.txt文件中的参数来生成不同的彩虹表

网站指纹识别

大多数是CMS，修改而来

whatweb+网址 扫描CMS，中间件，服务器，cookie，httponly，httpserver

发现CMS是joomla，谷歌针对该CMS工具，kali中有joomscan的工具

joomscan –url [网址] 用专门的工具扫描该CMS情况，有无防火墙，后台等

searchsploit joomla 3.7.0 搜索对应版本的漏洞

WAF识别

web application Firewalls

wafw00f [网址] 识别目标网站用了什么waf

一般国外的waf识别的比较准，国内的比较复杂，或者魔改过了，而且用工具比较浅显，用手工确认比较靠谱

nikto敏感目录扫描

nikto -h [目标网站] -o output.txt[默认输出到root目录下]

社工SET工具

setoolkit

里面选择社会工程学，网站克隆，可以结合ARP欺骗DNS劫持，即使输入正确的地址也是跳转到假界面。

社工获得登录名和密码

但是有时候不靠谱，可以选择里面手工网站模板

网站下载工具

一种爬虫，离线搜集信息的方式，配合用来制作钓鱼网站

wget命令，GNU开源项目中的一部分，主流win和linux都带有，用来下载的一个工具

wget [网址] 下载页面html

wget -r(递归下载) -p ./test(路径) [网址]

-t 设置重试次数。当连接中断（或超时）时，wget将试图重新连接。如果指
定-t0，则重试次数设为无穷多。

-T,–timeout=SECONDS 设置超时时间

-e “http_proxy=http://127.0.0.1:8087" 设置代理 https_proxy同理

-c 从断点出继续下载

-P 指定下载存放的目录

httrack

httrack [网址] 直接下载，只需要用下载目录中的网址文件夹就好了，比较好用

密码安全

medusa 服务密码破解

medusa -h [目标ip] -u test(用户名) -P（不知道的用大写，知道的用户名用小写u） mima.txt(root中的密码字典) -M ftp(破解ftp协议的密码)

hydra 服务协议密码破解

hydra 常用，支持众多协议

hydra -L [用户名字典] -P [密码字典] -t [线程] -vV（显示详细过程） -e ns（一般-e后面就加ns） [目标ip] [服务]

hydra -l 用户名 -p 密码字典 -t 线程 -o save.log -vV -e ns [目标ip] ssh 破解ssh 22号端口账户密码

hydra -l 用户名 -p 密码字典 -t 线程 -vV -e ns [目标ip] http-get /admin/login.php 破解网站指定页面的登录入口

hydra -l 用户名 -p 密码字典 xxx.pop3.mail pop3 破解pop3邮件

hydra -l boris(用户名) -P /usr/share/wordlists/fasttrack.txt 192.168.1.160 -s 55007（实验时非默认端口） pop3

远程桌面破解

hydra [目标ip] rdp -l administrator（一般win默认用户名） -P密码字典 -V

mysql密码

hydra -l root(mysql默认用户名) -p 密码字典 -e ns [目标ip] mysql 可以直接在网上破解3306端口

rar文件破解

kali中没有，apt-get install rarcrack 安装

rarcrack –type rar –threads 8 a.rar

破解a.rar的压缩包，提前先在a.rar目录下创建一个a.rar.xml的文件（新版kali不需要，会自动生成）

zip文件破解

fcrackzip -b -c[密码类型] -l 1-4（长度） -u [文件名.zip] -v

-c 密码类型： a=a-z 1=0-9 A=A-Z l=特殊字符

fcrackzip -b -c1 -l 1-4 -u b.zip -v
